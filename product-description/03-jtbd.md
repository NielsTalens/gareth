## Job Name:
Writing a new software feature

## Core Job Statement:
When writing new features, I want to make sure that they align with the goals of the product, so I can contribute to measurable outcomes instead of just shipping functionality.

## Desired Outcomes:
- Increase outcomes that contribute to objectives and make features explainable backed by metrics
- Minimize features that are not validatable 
- Reduce feature gluttony

## Emotional/Social Dimension:
- Feel contributing to the strategic direction
- Avoid feeling not being able to really prove features are worth it

## Current Frictions:
- Lack of strategic guidance
- Unclear understanding of the vision
- The work starts by adding stories in a backlog

## Existing Alternatives:
- LLM tools but it is a manual job

## Satisfaction (1–5): 5
